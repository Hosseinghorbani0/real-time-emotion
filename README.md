# Emotion Detection Using DeepFace

## 📌 Overview
This project detects facial emotions in real-time using a webcam and **DeepFace** for emotion analysis.

## 🚀 Features
- Real-time facial emotion recognition
- Uses **DeepFace** for accurate emotion detection
- Displays detected emotion on the screen
- Counts occurrences of **happy**, **sad**, and **angry** emotions

## 🛠️ Requirements
Make sure you have the following dependencies installed:

```bash
pip install opencv-python deepface
```

## ▶️ How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/hosseinghorbain0/emotion-detection.git
   cd emotion-detection
   ```
2. Run the script:
   ```bash
   python emotion_detection.py
   ```
3. Press **'q'** to exit the program.

## 📷 Output Example
The script captures real-time video and displays the detected emotion on the screen.
